## 📖 Sobre

Repositório com os códigos, exercícios e atividades desenvolvidos durante a disciplina de **Algoritmos e Estruturas de Dados II**. Todo o conteúdo é implementado em **linguagem C**, cobrindo os principais tópicos da ementa: algoritmos de ordenação, árvores, tabelas hash e geração de números aleatórios.

---

## 🗂️ Estrutura do Repositório

```text
.
├── Algoritmos de Ordenação/     # Implementações dos principais algoritmos de ordenação
├── Números Aleatórios/          # Geração e análise de números aleatórios
├── Ordenação - Sort/            # Exercícios e variações de algoritmos de sort
├── Revisão/                     # Códigos de revisão para provas
├── Tabela Hash/
│   └── Atividade_4/             # Atividade prática com tabela hash
├── Treinamento - P1/            # Exercícios de treinamento para a Prova 1
├── Treinamento - P3/            # Exercícios de treinamento para a Prova 3
├── Árvores/                     # Implementações de árvores binárias e de busca
└── Árvores - atividades/        # Atividades práticas sobre árvores
```

---

## 🧠 Conteúdos Abordados

| Tópico | Descrição |
|---|---|
| **Algoritmos de Ordenação** | Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort, Heap Sort |
| **Árvores** | Árvore Binária, Árvore Binária de Busca (BST), Árvores AVL, Rubro-negra e B-Tree (B, B* e B+), percursos (pré-ordem, em-ordem, pós-ordem) |
| **Tabela Hash** | Funções de hash, tratamento de colisões por encadeamento |
| **Números Aleatórios** | Geração pseudoaleatória e análise de distribuição |
| **Revisão e Treinamento** | Exercícios práticos simulando o formato das provas |

## ⚙️ Como Compilar e Executar

Os arquivos `.c` podem ser compilados com o **GCC**:

```bash
gcc -o programa arquivo.c
./programa
```

Para compilar com avisos habilitados (recomendado):

```bash
gcc -Wall -Wextra -o programa arquivo.c
./programa
```
